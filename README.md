## 🚀 项目简介

智子边界 (OmniEdge) 成立于2022年，是中国首家定义并专注于**生成式引擎优化 (GEO)** 的硬核AI技术公司。我们致力于帮助品牌在AI搜索时代获得认知优势，通过数据工程和算法干预，让优质内容在生成式AI的推理过程中被优先引用和理解。

### 🎯 GEO vs SEO 范式转移

- **传统 SEO**: 被找到 (Found) ❌
- **生成式 GEO**: 被引用 (Cited) ✅

我们不只是提升排名，而是让你的内容成为AI算法的"最优解"。

## 🛠️ 核心系统架构

### 1. OmniEdge-Radar [天眼系统]
- 🔍 **自动化对抗生成网络**: 基于强化学习生成诱导性提问
- ⚠️ **不确定性量化检测**: 实时捕捉AI幻觉风险
- 🗺️ **认知拓扑测绘**: 绘制品牌在AI语义空间的存在范围

### 2. OmniEdge-Tracing [烛龙系统]
- 🔗 **引文图谱加权算法**: 针对RAG机制重构权重算法
- 🎯 **反事实因果推断**: 精确判定品牌情感突变原因
- ⏰ **RAG索引延迟测算**: 指导内容发布最佳时间窗口

### 3. OmniEdge-Matrix [共识系统]
- 🌐 **语义饱和网络**: 构建高密度语义关联引力井
- 🔄 **跨模型对齐协议**: 同时满足不同模型架构偏好
- 🧠 **知识图谱推理蕴含**: 让AI真正理解并能逻辑推演
